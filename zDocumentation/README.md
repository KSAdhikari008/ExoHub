---

## Project Overview

The Smart Exhibition & Trade Fair Management System is a full-stack web application designed to simplify the management of exhibitions and trade fairs. It enables organizers to create and manage events, exhibitors to book booths, visitors to register for exhibitions, and administrators to oversee the entire platform through role-based dashboards.

The application is built using the MERN stack (MongoDB, Express.js, React.js, and Node.js) and follows a RESTful architecture with secure JWT-based authentication.

## Project Objective

The objective of this project is to develop a secure and scalable exhibition management platform that digitizes the workflow involved in organizing exhibitions and trade fairs.

The system allows:

* Administrators to manage events, booths, users, and bookings.
* Exhibitors to reserve and manage exhibition booths.
* Visitors to register for exhibitions and receive QR-based entry tickets.
